# 🚀 SmartAir AI

An AI-powered Air Quality Monitoring System that combines IoT sensors, Machine Learning, and a smart chatbot interface to provide real-time environmental insights.

---

## 🌟 Features

- 📡 Real-time Air Quality Monitoring (PM, Gas, Temperature, Humidity)
- 🤖 AI Chat Assistant with Voice Support
- 📊 Live Dashboard with Graph Visualization
- 📈 Machine Learning Prediction (Linear Regression)
- ⚠️ Smart Alerts (Gas Leak, High AQ, Health Risk)
- 🔐 Login System with Session Handling
- 📱 PWA Support (Install as App)

---

## 🧠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Charts: Chart.js
- AI Logic: Rule-based + ML Prediction
- Backend (Optional): Flask + SQLite
- IoT: Arduino / ESP32 Sensors

---

## 📊 ML Model

Implemented Linear Regression to predict future air quality:

- Uses historical AQ data
- Calculates trend + prediction
- Displays future AQ in dashboard

---

## 🖥️ Pages

- 🔑 Login Page
- 📊 Dashboard (Live Data + Graph)
- 🤖 AI Assistant (Chat + Voice)
- 📈 Prediction Page
